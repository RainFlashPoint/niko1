# 信贷流量平台 - 功能规格说明（纯卖流量版）

## 核心业务：流量分发

### 已删除
- ❌ 贷后管理（还款、催收）- 不放贷不需要
- ❌ 资金分润 - 只是卖线索，按条收费

### 保留功能

#### 1. 流量采购（上游）
- 渠道管理
- 流量数据明细
- API 配置

#### 2. 流量分发（核心）
- 进件管理（收到的线索）
- 线索筛选/打标签
- 分发给下游

#### 3. 流量销售（下游）
- 客户管理
- API 密钥
- 推送记录
- 计费配置（按条/按效果）

#### 4. 产品管理
- 合作产品展示
- 产品对接配置

#### 5. 数据报表
- 流量统计
- 转化分析
- 财务报表

#### 6. H5 落地页
- 信贷申请表单（接收线索）
