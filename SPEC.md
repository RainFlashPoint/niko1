# 流量平台管理系统 - Demo 需求说明

## 项目概述
- **项目名称**: 信贷流量服务平台
- **类型**: 后台管理系统 + H5 落地页
- **核心功能**: 流量采购、销售、报表、财务管理
- **目标用户**: 流量运营人员

## 页面结构

### 1. 管理后台 (Vue3 + Element Plus)

#### 1.1 首页/仪表盘
- 核心指标卡片：今日流量、转化率、收入、成本、利润
- 流量趋势图（近7天）
- 实时数据播报

#### 1.2 流量采购（上游管理）
- 上游渠道列表
- 流量数据明细
- API 配置管理
- 落地页管理

#### 1.3 流量销售（下游管理）
- 下游客户列表
- API 密钥管理
- 推送数据记录
- 客户使用报表

#### 1.4 数据报表
- 流量统计（来源/去向）
- 转化分析
- 财务对账

#### 1.5 财务管理
- 收入明细
- 成本记录
- 利润分析

### 2. H5 落地页
- 信贷申请落地页（模拟上游推送数据）
- 提交表单：姓名、手机、额度、地区等

## API 设计

### 上游调用接口（接收流量）
```
POST /api/ inflow
{
  "source": "上游渠道ID",
  "phone": "手机号",
  "name": "姓名",
  "amount": "期望额度",
  "area": "地区",
  "timestamp": "时间戳"
}
```

### 下游调用接口（推送数据）
```
POST /api/outflow
{
  "clientId": "客户ID",
  "phone": "手机号",
  "data": {...},
  "timestamp": "时间戳"
}
签名: MD5(clientId + timestamp + secret)
```

- 金融科技感：深蓝 + 青色主色调
- 卡片式布局
- 数据可视化图表
