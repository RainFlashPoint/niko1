二、撞库接口（MD5）

2.1 单条撞库

POST /api/v1/query/collision

请求头

参数说明
Content-Typeapplication/json
X-App-Id渠道标识
X-SignatureMD5签名

请求体

{
  "idCard": "110101199001011234",
  "phone": "13800138000",
  "name": "张三"
}

字段说明

字段类型必填说明
idCardstring✅身份证号（MD5加密传输）
phonestring✅手机号（MD5加密传输）
namestring✅姓名（MD5加密传输）

响应

{
  "code": 0,
  "message": "success",
  "data": {
    "queryId": "Q20240320001",
    "result": {
      "totalCount": 3,
      "platforms": [
        {"name": "平台A", "count": 2, "overdueCount": 0},
        {"name": "平台B", "count": 1, "overdueCount": 1}
      ],
      "riskLevel": "中风险"
    },
    "queryTime": "2024-01-15T14:30:00Z"
  }
}

三、进件接口（RSA）

注意：RSA公钥由平台预置提供，请联系管理员获取。

3.1 进件提交

POST /api/v1/entry/create

请求头

参数说明
Content-Typeapplication/json
X-App-Id渠道标识
X-SignatureMD5签名

请求体（RSA加密）

{
  "encryptedData": "BASE64编码的RSA加密数据"
}

加密数据结构

{
  "basicInfo": {
    "name": "张三",
    "idCard": "110101199001011234",
    "phone": "13800138000",
    "gender": "男",
    "age": 35
  },
  "financeInfo": {
    "income": "20000",
    "socialSecurity": true,
    "housingFund": true
  },
  "loanInfo": {
    "amount": "50000",
    "purpose": "经营周转",
    "term": 12
  }
}

响应

{
  "code": 0,
  "message": "success",
  "data": {
    "entryId": "E20240320001",
    "status": "pending",
    "createTime": "2024-01-15T14:30:00Z"
  }
}

3.2 状态说明

status说明
pending待审核
reviewing审核中
approved已通过
rejected已拒绝
disbursed已放款

四、加密说明

4.1 MD5 撞库加密

MD5(身份证号 + 手机号 + 姓名 + AppSecret)

4.2 RSA 进件加密

RSA公钥加密 + Base64编码

五、签名算法

Signature = MD5(请求体JSON + AppSecret)

六、错误码

code说明
0成功
1001缺少参数
1002格式错误
1003签名错误
1004渠道禁用
1005频率限制
2001加密失败
2002解密失败
